Requirements

  • from Aquamacs 3.3 on: Apple Mac OS X 10.9 or later.
  • from Aquamacs 3.0 on: Apple Mac OS X 10.6 or later, and (for the binary release) 64-bit Intel CPU (all modern Macs)
  • from Aquamacs 2.0 on: Apple Mac OS X 10.5.8 or later (Intel/PPC)
