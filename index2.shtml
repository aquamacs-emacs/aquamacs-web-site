Project News

Aquamacs 2.4 released. 2011-10-26
Version 2.4 has improved support for OS X "Lion" technologies.

Aquamacs 2.3 released. 2011-07-28
Version 2.3 is fast, easy to use and comes with updated support for hundreds of programming languages and file formats. It now supports OS X Services and can be used as external editor (ODB), and is even more reliable than before. The download package is smaller (Intel only, Universal Binary also available). Aquamacs 2.3 is based on the up-to-date GNU Emacs 23.3. The download is available at http://aquamacs.org/download.shtml. See the full changelog for details.
